FOGVISION ADAS - COMPREHENSIVE PROJECT SUMMARY
================================================

PROJECT OVERVIEW
----------------
Name: FogVision ADAS (AI-Powered Advanced Driver Assistance System)
Main File: test.py (~1760 lines, single-file Streamlit application)
Run Command: streamlit run test.py

TECHNOLOGY STACK
-----------------
- Language: Python
- UI Framework: Streamlit (futuristic dark automotive HMI theme)
- Computer Vision: OpenCV (cv2), NumPy
- Deep Learning: PyTorch, Ultralytics YOLOv8
- Maps: Folium, Streamlit-Folium
- Geocoding: Geopy (geodesic distance)
- Text-to-Speech: pyttsx3
- LLM: Ollama (qwen3:1.7b model)
- Hardware: CUDA GPU acceleration support


PROJECT STRUCTURE - 11 MODULES
------------------------------

1. VOICE AGENT
   - Priority-aware TTS system (HIGH/MEDIUM/LOW priorities)
   - Uses pyttsx3 for audio output
   - LLM response parser extracts structured fields
   - Cooldowns: HIGH=4s, MEDIUM=10s, LOW=20s
   - Threaded non-blocking operation
   - Transcript log of last 5 utterances

2. FOG DENSITY ESTIMATION (Dark Channel Prior)
   - Input: Raw frame (numpy array or file path)
   - Process: min(min(R,G,B)) over 15x15 window
   - Atmospheric light: mean of top 0.1% brightest dark channel pixels
   - Transmission: t = 1 - 0.95 * Dark(I/A)
   - Output: fog_density (0-100%), dark_channel visualization, transmission_map

3. IMAGE DEHAZING (Dark Channel Prior)
   - Class: DehazeModel
   - Parameters: window_size=15, omega=0.95, t0=0.1, refine=True
   - Optional guided filter refinement (opencv-contrib)
   - Output: Enhanced/cleared frame (uint8)

4. OBJECT DETECTION (YOLOv8)
   - Model: yolov8n.pt (nano variant)
   - Detects: cars, trucks, buses, motorcycles, traffic lights
   - Traffic light color: HSV-based RED/GREEN/YELLOW detection
   - Distance: 50 / object_height_pixels (in meters)
   - Confidence scoring per detection

5. RED GLOW / BRAKE-LIGHT DETECTION
   - HSV color space: two red ranges (0-10, 160-180 hue)
   - Minimum contour area: 300 pixels
   - IoU matching with vehicle boxes (threshold: 0.05)
   - Output: Boolean red_glow, glow_boxes, glow_count

6. ROAD CONTEXT (Geospatial)
   - Reverse geocoding: OpenStreetMap Nominatim API
   - Blackspot database: config/blackspots.json
   - Hazard types: fog_prone, rain_prone, landslide_prone, curve, high_speed
   - Nearby blackspot: < 2.0 km radius
   - Default location: 28.6139°N, 77.2090°E (Delhi)

7. RISK SCORE ENGINE
   - Weighted multi-factor formula:
     * Fog: 30%, Distance: 25%, Speed: 20%, Road: 10%, Red Glow: 10%, Humidity: 5%
   - Normalization ranges:
     * Fog: 0-100, Distance: 5-60m, Speed: 20-80 km/h
   - Hard overrides (force score >= 0.70):
     * Red glow + vehicle < 20m
     * Vehicle < 10m
     * Curve + vehicle < 25m
   - Classification: LOW (<=0.30), MEDIUM (0.30-0.60), HIGH (>0.60)

8. LLM INTEGRATION (Ollama)
   - Model: qwen3:1.7b (configurable to deepseek-r1:1.5b)
   - System prompt defines ADAS role and rules
   - Outputs: Risk Level, Hazard Alert, Recommended Speed, Driving Suggestion, Short Explanation
   - Gated interval: 5 seconds between inferences

9. VIDEO ENGINE
   - Samples exactly 1 frame per second
   - Source FPS: detected from video file, default 30.0
   - skip_frames = round(source_fps) - 1
   - Auto-loop for video files
   - Target size: 960x540 (default), resize to 640x480 in pipeline

10. PIPELINE ORCHESTRATOR
    - Caches: road_context (10s), LLM response (5s), fog_data
    - Frame processing: resize → fog density → dehaze → object detect → red glow
    - Rule engine generates safety alerts
    - Integrates voice agent for audio output

11. STREAMLIT DASHBOARD
    - Dark automotive HMI theme with custom CSS
    - Live processed video feed (dehazed + annotated)
    - Safety metrics tiles: fog density, visibility, recommended speed, nearest object, red glow
    - Risk score visualization with component bars
    - Geospatial map (CartoDB dark_matter) with blackspot markers
    - AI recommendation panel
    - Voice agent status and transcript
    - Detection manifest table

INPUT/OUTPUT
------------
INPUT:
- Video file: MP4, AVI, MOV, MKV (via file uploader)
- GPS location: streamlit_js_eval (browser geolocation) or default Delhi coordinates

OUTPUT:
- Real-time video with YOLO annotations and red glow highlights
- Fog density percentage and visibility rating
- Risk score (0-100) and risk level (LOW/MEDIUM/HIGH)
- Safety alerts (critical/warning/info)
- AI-generated driving recommendations via LLM
- Voice audio alerts for critical conditions
- Geospatial map with nearby blackspots


FORMULAS REFERENCE
-------------------

Dark Channel Prior (Dehazing):
- Dark(x) = min over local window (min(R, G, B))
- A = mean of top 0.1% brightest pixels in dark channel
- t(x) = 1 - 0.95 * Dark(I(x) / A)
- J(x) = ((I(x) - A) / t(x)) + A

Fog Density:
- fog_density = mean(Dark Channel) * 100

Distance Estimation:
- distance = 50 / max(height_pixels, 1)

IoU (Intersection over Union):
- IoU = intersection / union
- intersection = overlap_width * overlap_height

Risk Score:
- risk = 0.30*fog + 0.25*distance + 0.20*speed + 0.10*road + 0.10*red_glow + 0.05*humidity

Recommended Speed:
- fog >= 80% → 30 km/h, >= 60% → 40 km/h, >= 40% → 50 km/h, >= 20% → 60 km/h, else → 70 km/h
